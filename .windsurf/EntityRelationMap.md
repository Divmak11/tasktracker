## 2. Collection Schemas

### 2.1 Users Collection
**Path:** `/users/{userId}`

| Field | Type | Description | Constraints |
|-------|------|-------------|-------------|
| `id` | string | Firebase Auth UID | PK, unique |
| `name` | string | Display name | Required |
| `email` | string | Email address | Required, unique |
| `role` | enum | `super_admin` \| `team_admin` \| `member` | Required |
| `status` | enum | `pending` \| `active` \| `revoked` | Required |
| `teamIds` | string[] | Team memberships | FK â†’ teams.id |
| `fcmToken` | string? | Push notification token | Optional |
| `googleCalendarConnected` | boolean | Calendar OAuth status | Default: false |
| `googleAccessToken` | string? | OAuth access token | Encrypted |
| `googleRefreshToken` | string? | OAuth refresh token | Encrypted |
| `createdAt` | Timestamp | Account creation | Auto |
| `lastActive` | Timestamp | Last activity | Auto-updated |
| `approvedBy` | string? | Admin who approved | FK â†’ users.id |
| `approvedAt` | Timestamp? | Approval time | Optional |
| `avatarUrl` | string? | Profile image URL | Optional |
| `notificationPrefs` | object? | Notification settings | Optional |

### 2.2 Teams Collection
**Path:** `/teams/{teamId}`

| Field | Type | Description | Constraints |
|-------|------|-------------|-------------|
| `id` | string | Auto-generated | PK |
| `name` | string | Team name | Required, unique |
| `adminId` | string | Team admin | FK â†’ users.id |
| `memberIds` | string[] | Team members | FK â†’ users.id |
| `createdBy` | string | Creator (Super Admin) | FK â†’ users.id |
| `createdAt` | Timestamp | Creation time | Auto |
| `updatedAt` | Timestamp? | Last update | Auto |
| `updatedBy` | string? | Last modifier | FK â†’ users.id |

### 2.3 Tasks Collection
**Path:** `/tasks/{taskId}`

| Field | Type | Description | Constraints |
|-------|------|-------------|-------------|
| `id` | string | Auto-generated | PK |
| `title` | string | Task title | Required, max 100 |
| `subtitle` | string | Description | Max 500 |
| `assignedType` | enum | `member` \| `team` | Required |
| `createdBy` | string | Task creator | FK â†’ users.id |
| `status` | enum | `ongoing` \| `completed` \| `cancelled` | Required |
| `deadline` | Timestamp | Due date/time | Required, future |
| `createdAt` | Timestamp | Creation time | Auto |
| `updatedAt` | Timestamp | Last update | Auto |
| **Legacy Single-Assignee** | | | |
| `assignedTo` | string? | Single user ID | FK â†’ users.id |
| `calendarEventId` | string? | Google Calendar ID | Optional |
| `completedAt` | Timestamp? | Completion time | Optional |
| `completionRemark` | string? | Completion note | Max 500 |
| **Multi-Assignee** | | | |
| `isMultiAssignee` | boolean | Multiple assignees flag | Default: false |
| `assigneeIds` | string[] | All assignee IDs | FK â†’ users.id |
| `supervisorIds` | string[] | Can view all status | FK â†’ users.id |
| `sourceTeamId` | string? | Original team | FK â†’ teams.id |
| `taskGroupId` | string? | Links related tasks | Optional |

### 2.4 Assignments Subcollection
**Path:** `/tasks/{taskId}/assignments/{assignmentId}`

| Field | Type | Description | Constraints |
|-------|------|-------------|-------------|
| `id` | string | Auto-generated | PK |
| `userId` | string | Assigned user | FK â†’ users.id |
| `status` | enum | `ongoing` \| `completed` | Required |
| `assignedAt` | Timestamp | Assignment time | Auto |
| `completedAt` | Timestamp? | User's completion time | Optional |
| `completionRemark` | string? | User's remark | Max 500 |
| `calendarEventId` | string? | User's calendar event | Optional |

### 2.5 Remarks Collection
**Path:** `/remarks/{remarkId}`

| Field | Type | Description | Constraints |
|-------|------|-------------|-------------|
| `id` | string | Auto-generated | PK |
| `taskId` | string | Parent task | FK â†’ tasks.id |
| `userId` | string | Author | FK â†’ users.id |
| `message` | string | Comment text | Required, max 300 |
| `createdAt` | Timestamp | Creation time | Auto |

### 2.6 Approval Requests Collection
**Path:** `/approvalRequests/{requestId}`

| Field | Type | Description | Constraints |
|-------|------|-------------|-------------|
| `id` | string | Auto-generated | PK |
| `type` | enum | `reschedule` \| `user_access` | Required |
| `requesterId` | string | Who requested | FK â†’ users.id |
| `targetId` | string | Task or User ID | FK |
| `payload` | object | Request details | Varies by type |
| `status` | enum | `pending` \| `approved` \| `rejected` | Required |
| `approverId` | string? | Who resolved | FK â†’ users.id |
| `createdAt` | Timestamp | Request time | Auto |
| `resolvedAt` | Timestamp? | Resolution time | Optional |

### 2.7 Invites Collection
**Path:** `/invites/{inviteId}`

| Field | Type | Description | Constraints |
|-------|------|-------------|-------------|
